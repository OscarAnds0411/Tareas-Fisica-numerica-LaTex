# Tareas de Física Numérica en LaTeX

Repositorio que contiene las tareas del curso de **Física Numérica**, desarrolladas en **LaTeX**.

## Contenido
- Archivos `.tex` de cada tarea.
- Recursos e imágenes usados en las tareas.
- PDFs generados tras la compilación.

## Requisitos
- Distribución LaTeX (TeX Live, MiKTeX, etc.).
- Paquete `minted` y `Pygments` instalados para el resaltado de código.
- Editor compatible (VS Code, TeXstudio, Overleaf, etc.).
